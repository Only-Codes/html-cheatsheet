Welcome To My HTML Cheatsheet

This cheatsheet can be used to quickly remember different HTML tags

Document Structure

These HTML tags define the structure of the document

HTML Tag Name Description
<html> HTML Defines the root of an HTML document
<head> Head Contains meta-information about the HTML document
<title> Title Specifies the title of the HTML document
<body> Body Contains the content of the HTML document

Text Formatting

These HTML tags define specific text styles

HTML Tag Name Description
<h1> - <h6> Heading 1 - Heading 6 Defines the headers (headings) of different levels
<p> Paragraph Defines a paragraph
<strong> Strong Represents strong importance or bold text
<em> Emphasis Represents emphasised text or italic text
<u> Underline Represents underlined text

Lists

These HTML tags define how to create lists.

HTML Tag Name Description
<ul> Bulleted List Creates an unordered (bulleted) list
<ol> Numbered List Creates an ordered (numbered) list
<li> List Item Defines a list item

Links & Navigation

These tags are used to link different things and define a navigation

HTML Tag Name Description
<a> Hyperlink Defines a hyperlink
<nav> Navigation Defines a navigation bar

Images

This section defines the image tag

HTML Tag Name Description
<img> Image Embeds an image onto the page

Tables

These HTML tags are used to create tables

HTML Tag Name Description
<table> Table Defines a table
<tr> Table Row Defines the row of a table
<th> Table Head Defines the header cell of the table
<td> Table Data Defines a standard cell in the table

Forms

These tags are used to create forms

HTML Tag Name Description
<form> Form Defines an HTML form for user input
<input> Input Creates an input field which can have multiple input types
<textarea> Text Area Defines a multi-line text input field
<button> Button Defines a clickable button

Semantic Elements

HTML tags to define the structure of web content

HTML Tag Name Description
<div> Div Defines a division or section in an HTML document
<span> Span Defines an inline container for text
<header> Header Define structural element for a group of navigation elements
<footer> Footer Defines the footer section of the page
<article> Article Creates a standalone piece of content
<section> Section Defines a grouping of content with the page
<main> Main Represents the main content of the document
<aside> Aside Represents content aside from the content it is placed in
